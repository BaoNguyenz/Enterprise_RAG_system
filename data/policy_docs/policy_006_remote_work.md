# Remote Work and Hybrid Work Policy
**Document Type:** Policy
**Policy ID:** POL-006
**Owner:** HR / IT Department
**Effective Date:** 2024-02-01

## Purpose
Establish guidelines for employees working remotely to ensure productivity, security, and compliance with company standards.

## Stakeholders
| Role | Responsibility |
|------|---------------|
| HR Department | Policy administration, eligibility decisions |
| IT Department | Device provisioning, VPN access |
| IT Security Team | Endpoint security, remote access monitoring |
| Managers | Approve remote work arrangements, performance oversight |
| Employees | Comply with policy, maintain secure workspace |
| Finance | Home office stipend processing |

## Eligibility
Remote work is available to employees who:
- Have completed 90-day probationary period
- Have a role compatible with remote work (approved by manager)
- Maintain satisfactory performance rating
- Have a secure, dedicated workspace

## Security Requirements
Employees must:
- Use company-issued device OR personally-owned device enrolled in MDM
- Connect via TechDocs VPN for all internal system access
- Enable full-disk encryption on all work devices
- Use password manager (approved: 1Password or Bitwarden)
- Not use public WiFi without VPN enabled
- Lock screen when stepping away

## Home Office Equipment
| Item | Company Provided | Stipend Available |
|------|-----------------|------------------|
| Laptop | Yes (primary) | No |
| Monitor | No | $300 one-time |
| Chair | No | $200 one-time |
| Internet | No | $50/month |
| Webcam/Headset | Yes (on request) | No |

## Incident Reporting
If a work device is lost or stolen, report immediately to:
1. IT Security: security@techdocs.com
2. Direct manager
3. HR (for company property tracking)
